# URL Shortener

> Redirecting URLs

## File structure

- [src](src)
  - [index.ts](src/index.ts) - API routes defined with [Hono](https://hono.dev/)

## Setup

Using [bun](https://bun.sh/):

```
bun i
```

## Run

```
bun dev
```

## Deploy

Deploy on [Cloudflare Workers](https://hono.dev/getting-started/cloudflare-workers):

```
bun deploy --name <name-of-project>
```
